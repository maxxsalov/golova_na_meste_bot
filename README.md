# А голову не забыл?

Telegram-бот-напоминалка для двоих. Живёт прямо в вашем общем чате — пишите как человеку, он поймёт.

## Как начать

1. Добавьте бота в групповой чат (вы + партнёр + бот)
2. Каждый участник представляет себя:
   - `/reg Антон`
   - `/reg Маша`
3. Готово — просто пишите в чат

## Что понимает

Бот слушает сообщения со словами «напомни», «через N минут/час», «завтра в HH:MM» и автоматически создаёт напоминание.

| Вы пишете | Что произойдёт |
|---|---|
| `напомни мне через 30 мин позвонить` | Напомнит вам через 30 минут |
| `напомни ей завтра в 9:00 купить хлеб` | Напомнит партнёрше завтра в 9:00 |
| `напомни ему позвонить в банк` | Напомнит партнёру (по умолчанию через 30 мин) |
| `через час выключить духовку` | Напомнит вам через час |
| `напомни Антону созвониться` | Напомнит Антону (по имени) |

Если время не указано — бот напомнит через 30 минут.

## Уведомления

Когда приходит время, бот пишет в чат:

```
Антон, 🔔 МАША ПРОСИЛА НАПОМНИТЬ: выключить духовку

[✔️ Сделано]  [⏰ Ещё через 10 мин]
```

- **Сделано** — бот напишет «Антон выполнил напоминание»
- **Ещё через 10 мин** — напомнит ещё раз

## Команды

| Команда | Описание |
|---|---|
| `/reg Антон` | Представиться |
| `/my` | Мои активные напоминания |
| `/our` | Напоминания от партнёра |
| `/del 42` | Удалить напоминание по ID |
| `/tz 3` | Установить часовой пояс (UTC+3 — Москва) |
| `/quiet 2ч` | Не беспокоить 2 часа |
| `/who` | Как меня зовёт бот |
| `/stats` | Статистика за 7 дней |
| `/faq` | Частые вопросы |
| `/help` | Справка |
